Membangun Desain Modern Langsung di HTML Anda

1. Filosofi "Utility-First"

Tidak seperti Bootstrap yang memberikan komponen jadi (seperti `.card` atau `.btn`), Tailwind menyediakan kelas-kelas "utility" tingkat rendah yang masing-masing melakukan satu hal spesifik.

Contoh:

  • `p-4`: Memberi padding sebesar 1rem di semua sisi.
  • `text-center`: Membuat teks rata tengah.
  • `font-bold`: Menebalkan teks.
  • `bg-red-500`: Memberi warna latar belakang merah.

Dengan menggabungkan utility ini, Anda bisa membangun desain kustom tanpa menulis CSS sama sekali.

3. Keuntungan & Keterbatasan CDN

Keuntungan:

  • Sangat cepat untuk memulai dan prototyping.
  • Tidak perlu proses build atau instalasi.

Keterbatasan:

  • Tidak bisa mengkustomisasi tema default Tailwind (warna, spacing, dll).
  • Tidak bisa menggunakan directive seperti `@apply`.
  • Ukuran file lebih besar karena men-generate CSS di browser.

Untuk proyek produksi, disarankan menggunakan Tailwind CLI atau PostCSS untuk performa dan kustomisasi terbaik.

Overflow Property

Overflow Visible (Default)

Content that overflows the size of the container will still show with overflow : visible (Default behavior).

Overflow Hidden

Content that overflows the size of the container will be cut with overflow : hidden

Overflow Scroll

Content that overflows the size of the container can be scrolled with overflow : scroll. Scrollbar will always be shown.

Overflow Auto

Content that overflows the size of the container can be scrolled with overflow : auto. Scrollbar will only be shown if needed
